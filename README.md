# Cursorphobic

![alt text](https://raw.githubusercontent.com/sambernhardt/cursorphobic/master/demo.gif "Logo Title Text 1")

## Usage
```javascript
cursorphobic('.selector')
```

### Configurations
```javascript
cursorphobic('.selector', {
  proximity: 50
})
```

key | option type | example
----|---------|------
`proximity` | integer | `50`
